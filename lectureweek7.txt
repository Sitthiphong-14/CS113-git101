git ไงจะใครหล่ะ
คำสั่ง
__________________________________________________
                                 |__บน Command prompt_|__Terminal/Bash_|
สร้าง directory |       mkdir      |      mkdir     |
--------------------------------------------------
ดูรายการ direcและ |                    dir        |       ls       |
files       |                   |                |
--------------------------------------------------
ดูแบบละเอียด             |                     dir        |       ls -l    |
--------------------------------------------------
git --version แสดง version git ปัจจุบันของเครื่อง
------------------------------------------------------------------
git init ใส่ git เข้าไปใน directory
------------------------------------------------------------------
git status เช็ค status ของไฟล์
------------------------------------------------------------------
git add เพิ่มไฟล์เข้าไป พร้อมจะ commit
------------------------------------------------------------------
git config user.name && git config user.email ตั่งค่าชื่อและอีเมล
"ถ้าเครื่องส่วนตัวมี --global หลัง config ด้วย"
------------------------------------------------------------------
git commit -m คือ commit ไฟล์ เพื่อเก็บ Version/Snapshot/Checkpoint (แล้วแต่จะเรียก)
                -m เป็นการกำหนดว่าเราทำไรไว้กับไฟล์
------------------------------------------------------------------
git log --all --decorate --oneline --graph ดู checkpoint ทั้งหมด
------------------------------------------------------------------
git remote add <ชื่อ remote เช่น origin> <git url> เชื่อม local reporitory
------------------------------------------------------------------
git push -u origin master ส่งไฟล์เข้า git
------------------------------------------------------------------
git checkout <commit-id> นำ checkpoint เก่ากลับมาทำงานทังหมด
git checkout <commit-id> <ชื่อไฟล์> บางไฟล์
------------------------------------------------------------------
เมื่อเปลี่ยนเครื่อง หรือลบ local repository ไปแล้ว อยากจะนำ git repository ที่เก็บไว้ใน server มาทำงานต่อ
ให้ใช้คำสั่ง 
git clone <git-url>
------------------------------------------------------------------
